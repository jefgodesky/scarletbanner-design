  • This component is best used for a menu that takes up most or all of a page (like the homepage for this design system).
  • The panel is designed to work with secondary, tertiary, and quarternary headings (<h2>, <h3> and <h4>), but not primary headings (<h1>). Each page should have only one primary heading, so if you’re considering using it in a panel, you might be using the wrong component.
  • Since the entire panel is a link, don’t use any other links or buttons inside the panel.

Design Tokens

Design tokens provide a way to record and distribute the atomic design decisions that form the foundation of a design system — things like colors, typefaces, and spacing choices.

The canonical source for our design tokens is our tokens file. This is a JSON file that follows the W3C Design Tokens Format Module. If you are using a tool that recognizes that format, you can use the link above to import our tokens directly.

Colors

If you’d like to use the SCSS variables provided for these tokens, you’ll need to import the _colors.scss module (found at stylesheets/src/modules/_colors.scss in our code repository after the tokens have been generated) with a line like the following at the top of your SCSS stylesheet:

@use "/path/to/repo/stylesheets/src/modules/_colors.scss"; {{ swatches }}

The Scarlet Banner

Design Tokens

Design tokens provide a way to record and distribute the atomic design decisions that form the foundation of a design system — things like colors, typefaces, and spacing choices.

The canonical source for our design tokens is our tokens file. This is a JSON file that follows the W3C Design Tokens Format Module. If you are using a tool that recognizes that format, you can use the link above to import our tokens directly.

Page Shadow

SCSS Variable {{ shadow.page.scss }}
CSS {{ shadow.page.css }}

This shadow is used on the main element and a few other key components tied to layout. It is deep, dark, and generally too much for standard UI components. Use sparingly and with caution.

Block Shadow

SCSS Variable {{ shadow.block.scss }}
CSS {{ shadow.block.css }}

This shadow is used on most UI elements. It provides a subtle impression of depth suitable for widespread use.

Pressed Shadow

SCSS Variable {{ shadow.pressed.scss }}
CSS {{ shadow.pressed.css }}

This shadow is identical to the block shadow, but with a reduced blur. Active buttons switch to this shadow to reinforce the physical sense of a button being pressed.

White Glow

SCSS Variable {{ shadow.white glow.scss }}
CSS {{ shadow.white glow.css }}

This shadow creates a white glow.

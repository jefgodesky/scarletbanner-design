Design Tokens

Design tokens provide a way to record and distribute the atomic design decisions that form the foundation of a design system — things like colors, typefaces, and spacing choices.

The canonical source for the design tokens in this system are the YAML files in the tokens directory of our source code repository. If you are using this design system for something other than a website, we encourage you to pull these values directly from there, so that your work will be automatically updated when new changes are introduced. This page documents the values in those YAML files in a way that is more useful and readable for humans.

This is the design system for the Scarlet Banner. You can find the source code for this system on Github. To use it on your own website, add the following to your HTML header:

<link rel="stylesheet" href="{{ cdn.root }}/style.css" crossorigin="" />

And the following at the end of your HTML file, just before the close of your <body> tag:

<script src="{{ cdn.root }}/loader.js"></script>

Design Tokens

Design tokens are the true atoms of a design system, the most basic design decisions that everything else is made of. Here you’ll find documentation for things like colors, typefaces, and spacing.

Typography

As Oliver Reichenstein said, web design is 95% typography. Here we document the basics of how we use typography in this design system and the visual hierarchy that the system is built upon.

Components

These are the repeatable components that pages are made up of in the Scarlet Banner design system, starting with the smallest and simplest “molecules” and building up to larger and more complex “organisms.”
